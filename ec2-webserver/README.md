# EC2 웹서버 구축 프로젝트

## 📌 목표
- EC2로 웹서버 구축
- Apache 설치
- 외부 접속 성공

## 🛠 사용 기술
- AWS EC2
- Amazon Linux
- Apache

## 📋 구축 과정

1. EC2 인스턴스 생성
2. 보안그룹 설정 (80, 22 포트)
3. Apache 설치
4. 서버 실행
5. 접속 확인

## ❗ 문제 & 해결

### 접속 안 됨
- 원인: 80포트 막힘
- 해결: 보안그룹 수정
